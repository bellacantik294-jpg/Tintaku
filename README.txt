README - Air Mata Kertas (All Features)
This package adds:
- Dark/light theme (toggle)
- Animations & nicer cards
- Admin: upload cover image (saved as data URL in localStorage)
- Likes per story (saved in localStorage)
- Comments via localStorage
- Floating Trakteer button & support popup
- All pages synced with pro-script.js

How to deploy:
1. Upload all files in this ZIP to your GitHub repository (replace existing files).
2. Ensure index.html, style.css, pro-script.js, cerpen-data.js are in repo root.
3. Open site and use Admin to add stories. When uploading a cover image in Admin, it will be saved alongside the story (stored in localStorage).
4. Likes/comments are stored locally on visitor's browser.
